Git Ng Web
==========

Installation
-------------

.. code-block:: bash

  cd where_i_want_to_install_the_package
  wget https://raw.githubusercontent.com/waxe/waxe-image/develop/install.sh
  sh ./install.sh


Start / stop the server
-----------------------

.. code-block:: bash

  ./git_ng_web.sh start
  ./git_ng_web.sh stop
  ./git_ng_web.sh restart
